# goctl (Non-official)
goctl is a cli tool that helps you to work with go-zero.

The registry can be found [here](https://hub.docker.com/r/jjkoh95/goctl).

## Usage
```bash
docker run --rm jjkoh95/goctl:v1.4.3 --version
# goctl version 1.4.3 linux/amd64
```
